# Pastebin Lite

A minimal Pastebin-like application built with Next.js and Vercel KV.

## Features
- Create text pastes
- Shareable URLs
- Optional TTL and view limits
- Deterministic expiry testing support

## Tech Stack
- Node.js (Next.js App Router)
- Vercel KV (Redis)

## Run Locally
```bash
npm install
npm run dev
